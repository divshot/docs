# AngularJS RouteProvider and LocationProvider HTML5 Mode

To use the AngularJS RouteProvider HTML5 mode, set up the routes in your [Configuration File](/guides/configuration) as seen below:

```json
{
  "name": "my-app-name",
  "routes": {
    "**": "index.html",
  }
}
```
