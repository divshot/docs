Divshot Architect

These guides will give you a quick overview of the various features of Divshot Architect and how they can be used to quickly build professional quality HTML.

Interface Overview

This guide provides a high-level introduction to the various parts of Divshot's interface including the canvas, component drawer, inspector, page and folder dropdowns, and more.

Folders and Pages

Learn how Divshot's folders and pages can be used to organize your work as well as how to create re-usable templates.

Working With Components

Components are the building blocks of every page. Learn how to add components to the canvas, configure their properties, make them responsive, and customize their look and feel using custom classes and IDs.

Editing Content

Learn how to edit content on the canvas, including modifying text, dragging and dropping components, cut, copy, and paste, and how to navigate your selection using breadcrumbs.

Using the Source Editor

In addition to Divshot's powerful visual editing capabilities, it also ships with full-featured code editing. Learn how and when to use the source editor to add custom styles, modify content, or add JavaScript behaviors to your pages.

Preview and Export

Divshot allows you to export pages or entire folders to clean HTML and CSS. Learn how to use the preview and export functionality of Divshot to share your work with those who need to see it.
